# Predicciondeataquescerebrales

Titulo: Predictor de ataques cerebrales

Autores: Sebastian Andrade, Belen Begnini & André Sarmiento

Descripcion del proyecto:
El proyecto predice si una persona es mas propensa a tener un ataque cerebral que otra. El proyecto inicia realizando un EDA de la base de datos para eliminar valores atipicos y arreglar valores faltanes. Se divide el dataframe en set de entrenamiento, prueba y validacion. Se balancea los datos mediante SMOTE y RandomUnderSampler, se realiza una prueba estadistica para ver que modelo es mejor. Se pone ese set balanceado dentro de diferentes modelos para ver que modelo obtiene un mejor resultado a la hora de predecir.

Archivos subidos:
- Jupiter Notebook con todo el código
- Base de datos para el código
